WELCOME TO GAME OF MUSIC

For this programm you need the java runtime environment

Put your music in the songs folder to make it accessible for the program.
Then you can choose them in the interface.

On the interface, you can change some values to fit the visualization to your music.
Press p to get back to that interface.

Example: https://vimeo.com/235226370

Values to Tweak:
Sound Reaction Speed: How fast the music reacts to the music
Glitch factor: for the high tones there is a glitch in the visualisation, turn that value up or down to get more or less glitch
Bass tolerance: Turn down so the visualisation reactes more to the bass or turn up so the music reacts less to the bass
Color Tweaks: Change the colors of the visualisation. Turn up one color or turn down another to not always get grey for example

Modes and how to switch through them (press the Button in the [])
Field Modes:
[1]...............normal mode
[2]...............obelisk mode (turn on the shine mode [press s] for cooler effect)
[3]...............greater smaller mode
[4]...............skyscraper mode
[5]...............cube mode

Color Modes:
[a]..............."normal" mode
[s]................shine mode
[d]................no border mode
[f]................subtract mode
[g]................flickering mode


[p]...............pause the song and get back to the interface (menu)

Used Libraries
Minim Library (http://code.compartmental.net/tools/minim/)
G4P (http://www.lagers.org.uk/g4p/)
